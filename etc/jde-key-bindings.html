JDE Key Bindings (Note: ESC and ALT are equivalent.)

Code Completion/Insertion
key binding generates
jde-checkstyle runs checkstyle on the current buffer
C-c C-v C-. jde-complete displays menu of completions for symbol at point
C-c C-v . jde-complete-in-line completes symbol at point. repeating cycles through potential completions
C-c C-v t jde-gen-try-catch-wrapper wrap the region into a try/catch block
C-c C-v f jde-gen-try-finally-wrapper wrap the region into a try/finally block
C-c C-v C-l jde-gen-println insert println statement
C-c C-v C-z jde-import-find-and-import an import statement for the class at point
C-c C-v j jde-javadoc-autodoc-at-line update javadoc comment block for declaration at current line
jde-wiz-delegate methods that delegate tasks to a specified class
C-c C-v e jde-wiz-extend-abstract-class implementations of abstract methods inherited by class containing point
jde-wiz-get-set-methods Get and set methods for the private fields of the class containing point
C-c C-v i jde-wiz-implement-interface implementation of an interface in the class containing point
C-c C-v o jde-wiz-override-method a method that overrides a method inherited by the class containing point
